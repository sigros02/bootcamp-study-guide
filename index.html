Prework Study Guide

HTML

  • head tag is invisible to the user and defines how information is handled by the client for this web page
  • meta tag is invisible to the user and holds variables and values that are used by the client for this wepage
  • title tag is someitme used to define the search engine tab name which is visible to the user. More importantly it is used for search engine optimization

CSS

  • INLINE CSS: Styles element directly in HTML file. Applies to specific HTML tag. Consists of a "style" attribute follwoed by a CSS rule.
  • INTERNAL STYLE SHEET: Contains CSS rules for the webpage in the head section of the HTML file. The rules only apply to that webpage
  • EXTERNAL STYLE SHEET: Links a file using "link" element containing CSS rules separatley. Makes design changes easier.
